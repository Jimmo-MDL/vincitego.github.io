VincitEgo's Idle Heroes Simulators

PvP Simulator

Simulates PvP combat so you can test out different heroes and team setups.

Multiple Team Testing

** Currently out of comission due to code refactor ** Uses the PvP Simulator to test multiple teams at once, pitting them against each other.

PvP Genetic Algorithm

Uses the PvP Simulator to simulate natural selection to find the best overall team. Generates random teams, makes them fight each other, then breeds the top teams. Rinse and repeat. Currently pointing to multi-threaded web worker version for speed.

PvP Genetic Targeting

** Currently out of comission due to code refactor ** Uses the PvP Simulator to simulate natural selection to find a team to beat a target team. Generates random teams, makes them fight the target team, then breeds the top teams. Rinse and repeat.

Imp's Adventure

Plug in your current situation and it will run 100 thousand simulations. It will output a chart with how many stars you can expect to get. Use it towards the end of the event to see if spending gems or money for additional dice will be worth it.
